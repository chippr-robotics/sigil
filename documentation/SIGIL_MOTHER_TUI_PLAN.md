# Sigil Mother TUI - Comprehensive Implementation Plan

## Executive Summary

The Sigil Mother TUI provides a secure, intuitive terminal user interface for administrators managing MPC (Multi-Party Computation) signing operations. Built with **ratatui** (Rust TUI framework), it serves as the primary interface for disk lifecycle management, child account administration, reconciliation workflows, and secure reporting.

This interface protects the user's **life savings** - every design decision prioritizes security, clarity, and auditability.

### 2.2 Operation Authorization Levels

| Level | Operations | Requirements |
|-------|-----------|--------------|
| **L0** | View status, browse logs | PIN authenticated |
| **L1** | Sign transactions, view children | PIN authenticated |
| **L2** | Create child, refill disk | PIN + confirmation dialog |
| **L3** | Nullify child, export master shard | PIN + typed confirmation + warning |
| **L4** | Factory reset, change PIN | PIN + typed confirmation + countdown |
